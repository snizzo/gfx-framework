Gfx-framework

Small and easily hackable web framework

View on GitHub Download .zip Download .tar.gz Docs

No must-use paradigms

Gfx requests no paradigm to be used. Feel free to use MVC or singleton or just include a class or a portion of the framework, it will still work.

From little to big, from big to little

Scales from very little projects and utilities to big applications requiring speicific features (rewriting, database abstraction)

There's little to no under the hood

Gfx can be considered a useful extension to the php standard library. Every line of code you write, will exactly do what you're thinking it will do. There are no obscure mechanics you are tide of.

Easily hackable

Gfx is probably the easiest framework around to be hacked and adapted in very different scenarios.
